# Logastellus

**Seamlessly create, save and share your sound.**

Logastellus is a next-generation audio platform exploring a future where creators, communities and organisations have greater ownership over their content.

Built around the principles of openness, privacy and user sovereignty, Logastellus aims to make audio creation, storage and distribution more resilient, more portable and less dependent on centralised platforms.

🌐 **Live Prototype**
https://rc1732.github.io/logastellus/

## Current Status

🚧 Prototype / Early Development

This repository contains the first public prototype of Logastellus.

The project is currently focused on:

* Refining the user experience
* Validating the product concept
* Building a premium creator-focused interface
* Exploring decentralised storage workflows
* Investigating integration with the Logos ecosystem
* Establishing the technical foundation for future development

---

## Why Logastellus?

Most audio platforms rely on centralised infrastructure that controls how content is stored, distributed and monetised.

Logastellus explores a different approach.

We believe creators should have greater ownership over their content, greater control over how it is distributed and greater confidence that their work will remain accessible over time.

Our goal is to combine the simplicity of modern consumer software with the resilience of open and decentralised networks.

---

## Vision

Logastellus aims to become a modern audio platform where users can:

* Upload audio files
* Generate audio from text
* Store files using decentralised infrastructure
* Share content through secure links
* Maintain ownership of their data
* Operate their own storage infrastructure
* Participate in open, censorship-resistant networks
* Build independent audio archives
* Preserve digital content for the long term

## Running Locally

Clone the repository:

```bash
git clone https://github.com/RC1732/logastellus.git
```

Open the project folder:

```bash
cd logastellus
```

Launch:

```bash
open index.html
```

Or use a local development server of your choice.
